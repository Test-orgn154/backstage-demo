# Basket API

This documentation provides an overview of the Basket API, a microservice responsible for managing shopping basket data.

## Overview

The Basket API allows users to:
- Retrieve their shopping basket.
- Add, update, or remove items from their basket.
- Clear their basket.

## API Endpoints

For detailed API specifications, please refer to the OpenAPI documentation.

### Get Basket

`GET /api/v1/basket/{id}`

Retrieves the shopping basket for a given user ID.

### Update Basket

`POST /api/v1/basket/{id}`

Updates the shopping basket for a given user ID.

### Delete Basket

`DELETE /api/v1/basket/{id}`

Deletes the shopping basket for a given user ID.
